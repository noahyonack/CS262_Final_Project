parallelogram.config module¶

This file contains config values (like the port numbers) used by our library

parallelogram.config.MULTICAST_GROUP_IP = '224.3.29.71'¶

ip address of group to use for multicast communications

parallelogram.config.MULTICAST_PORT = 10000¶

port to use for multicast communications

parallelogram.config.PORT = 1001¶

port to use for socketcommunications
